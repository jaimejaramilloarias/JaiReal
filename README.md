# JaiReal-PRO

Editor de cifrados orientado a trabajo offline.

## Requisitos

- Node.js 18+

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

Visita <http://localhost:5173> en tu navegador.

## Pruebas

```bash
npm test
npm run test:e2e
```

## Lint

```bash
npm run lint
```

## Atajos de teclado

- `Ctrl+Shift+L`: alterna la visibilidad del renglón secundario.
